# 钉钉机器人 (DingTalk) 接入指南

钉钉机器人集成采用钉钉 Stream 模式，支持互动卡片实时展示，沟通体验更丰富。

## 1. 钉钉开放平台配置

### 1.1 创建应用
1. 登录 [钉钉开放平台](https://open-dev.dingtalk.com/)。
2. 点击 **“应用开发” -> “企业内部开发”**。
3. 点击 **“创建应用”**，选择“H5微应用”或“小程序”，填写信息并创建。

### 1.2 添加机器人能力
1. 在应用详情页，点击左侧导航 **“应用能力” -> “添加应用能力”**。
2. 找到 **“机器人”**，点击添加。
3. 在机器人设置页面，开启 **“机器人推送到群”** 和 **“机器人单聊”**（根据需求勾选）。

### 1.3 配置消息接收 (Stream 模式)
1. 在机器人设置页下方的 **“消息接收模式”** 中，选择 **“Stream 模式”**。
2. 点击保存。Stream 模式无需配置公网回调地址。

### 1.4 配置互动卡片 (重要)
1. 点击左侧导航 **“应用能力” -> “互动卡片”**。
2. **选择卡片类型**：点击按钮创建卡片，务必选择 **“AI 卡片”** 分类下的模板。AI 卡片专为流式输出和打字机效果设计。
3. 获取 **“模板 ID”**。或者在卡片搭建器中创建一个包含 `content` 字段的简单 AI 卡片。

### 1.5 获取凭据与权限
1. 在 **“凭据与基础信息”** 页面获取：
   - **Client ID** (原 AppKey)
   - **Client Secret** (原 AppSecret)
2. 在 **“开发配置” -> “权限管理”** 中，确保已拥有“企业内机器人发送消息”等相关权限。

---

## 2. CatWiki 后台配置

1. 进入 CatWiki 后台 **“站点设置” -> “机器人配置”**。
2. 选中 **“钉钉机器人”**，开启开关。
3. 填入：
   - **Client ID**
   - **Client Secret**
   - **模板 ID** (用于展示互动卡片)
4. 点击 **“保存”**。

---

## 3. 技术特性
- **同步频率**：0.8s / 次。
- **显示效果**：支持动态互动卡片，AI 回复过程中卡片会实时更新。
