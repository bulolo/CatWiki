外部网站模拟测试

此页面模拟了一个外部网站，用于测试 CatWiki 聊天挂件的嵌入效果。

挂件脚本加载自：http://localhost:8002/widget.js

挂件集成指南：

将以下代码嵌入到您的网站 HTML 中即可完成集成。挂件的所有表现（样式、内容、位置）均可在 CatWiki 管理后台统一配置。

<script src="http://localhost:8002/widget.js" data-site-id="1"></script>

工作原理：

  • data-site-id: 您的站点 ID。
  • 后端驱动内容： 挂件启动后，会自动从后端拉取该站点的 标题、主题色、欢迎语。
  • 前端控制布局： 挂件的物理显示位置（左/右/边距）完全由宿主页面控制，避免样式冲突。
  • 零闪烁加载： 挂件在配置拉取完成前保持隐藏，确保平滑淡入。

贴士：这种设计让您可以灵活地在不同页面通过 CSS 调整挂件位置，而核心内容则由管理后台统一维护。
